Welcome To Our Online Store

This website is created as part of our HTML assignment. It shows how different pages can be connected using links and how images can be displayed properly on web pages. This home page helps users navigate to different categories easily just like a real online shopping website.

Each category page contains images along with product names and prices. The purpose of this project is to practice HTML linking, image handling, and proper page structure.

Shop By Category

In the Ladies section, you will find products designed for women. The Gents section contains items for men. The Babies page includes products for kids and infants.The Home Appliances section shows items used in daily household life.

Click on any category from the navigation menu above to view products with images, names, and prices. All pages are connected using relative links for easy navigation.
